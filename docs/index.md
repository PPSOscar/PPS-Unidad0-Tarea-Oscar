# Puesta en Producción Segura  
## Unidad 0 – Tarea RA5

Bienvenido a la documentación oficial del proyecto realizado para la Unidad 0 del módulo **Puesta en Producción Segura**.

En este sitio web explico paso a paso cómo se ha desarrollado toda la actividad, desde la creación del repositorio hasta el despliegue final mediante Docker y NGINX.

---

## Objetivos de la práctica

- Utilizar Markdown de forma profesional.
- Trabajar con Git y GitHub para control de versiones.
- Crear un Workflow en GitHub Actions para generar documentación automática.
- Publicar una web en GitHub Pages.
- Levantar un contenedor NGINX que sirva la documentación localmente.
- Presentar evidencias claras del proceso.

## Resumen General

En esta práctica hemos recorrido un flujo completo de **DevSecOps básico**, donde:

- Se crea un repositorio correctamente estructurado.
- La documentación se escribe en Markdown y se transforma automáticamente en una web.
- Se automatiza la generación con GitHub Actions.
- Se publica online con GitHub Pages.
- Se replica el despliegue localmente usando un contenedor NGINX.

Cada apartado explica los comandos usados, configuraciones realizadas y problemas que se han resuelto durante el proceso.
