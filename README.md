## Puesta en Producción Segura

Repositorio correspondiente a la tarea de la Unidad 0 del módulo **Puesta en Producción Segura**, cuyo objetivo es aplicar conocimientos sobre Markdown, Git/GitHub, automatización mediante GitHub Actions, generación de documentación con MkDocs, publicación en GitHub Pages y despliegue básico con Docker y NGINX.

## Estructura del repositorio
- **docs/**: Documentación completa de la actividad.
- **calculator/**: Módulo Python de ejemplo incluido en la estructura solicitada.
- **.github/workflows/**: Workflow para la generación automática de documentación.
- **mkdocs.yml**: Configuración del sitio MkDocs.
- **requirements.txt**: Dependencias necesarias.
